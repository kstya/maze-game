🌀 Лабиринт
Игра-лабиринт на Python и Pygame — найди выход из запутанных коридоров.


🎮 Описание
Управляй персонажем, обходи стены и ищи выход из лабиринта. Реализована с использованием объектно-ориентированного программирования.


🛠️ Технологии

Python 3
Pygame



🚀 Запуск
bashpip install pygame
python main.py


🕹️ Управление

Стрелки — передвижение персонажа
R — перезапуск уровня
